Info

At the University of Missouri - St. Louis, the Information Systems department offers a course for Web Design. I took the course during the Summer Session which meant the class took 4 weeks to apply web methodologies.

The course focused on website planning, design, layout and construction. Topics covered include understanding site structure, navigation and content management, markup languages, CSS, JavaScript, and related web technologies, tools, libraries and frameworks.

Objectives of the course was to learn the web page layout and design, HTML5, , scripting languages & frameworks, Cascading Styling Sheets (CSS), and advanced web applications. The goal was to create a multiple-page site of the Grimm's Fairytale Stories.

The Challenge

In web design, you need to know where you want to locate items on the page and the overall end user experience. The biggest challenge for me was the creation behind the tax calculator and the discount. Both of these used ares in which I was just getting into. Overall, I learned a lot while developing these two pieces.

The Solution

To create a website which would allow an end-user the ability to read the books digitally, calculate taxes based on pre-defined percentages, and a store to calculate a discount based on the amount of books purchased based on price.

Responsibilities

Developing a front-end website utilizing HTML5, CSS, and JavaScript from start to finish. During the course, I was responsible for saving all work to Cloud9 which is now part of AWS.
