Info

At the University of Missouri - St Louis, one of the College of Business Administration basic foundation courses has a course number of IS 2800. That course is called Information Systems Concepts and Applications. The class has multiple assignments and one of them is to create a temperature converter.

The temperature converter is created by the students using JavaScript and HTML. As a graduate teaching assistant for the class, I decided to create the converter myself as well for some practice and to expand my knowledge.

The goal of this assignment is to provide the students the challenge of creating a front-end website.

The Challenge

As mentioned above in the info section, this is to challenge students of all levels to create a front-end website. The biggest challenge the students run into is knowing where to put the script, functions, and how to create a form. Personally, this was a refresher for me and my only challenge was the go back button.

The Solution

The solution is to create a front-end site in html in order for a end-user to get an end result of the temperature conversion.

Responsibilities

To develop a front-end html site with embedded JavaScript in order to meet the resolution.
